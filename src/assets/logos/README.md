# Logo Klasörü

Bu klasöre kartlarda kullanılacak uygulama logolarını ekleyin.

Önerilen kurallar:

- Mümkünse `svg` kullanın.
- Kare oranlı dosya kullanın.
- Şeffaf arka plan tercih edin.
- Dosya adlarını küçük harf ve kısa tutun.
- Boşluk yerine `-` kullanın.

Örnek dosya adları:

- `metamask.svg`
- `wordpress.svg`
- `gmail.svg`
- `steam.svg`
- `instagram.svg`
- `discord.svg`
- `microsoft.svg`
- `lol.svg`
- `valorant.svg`
- `tft.svg`
- `ea.svg`
- `facebook.svg`

Not:
Sonraki adımda bu dosyaları bir logo kataloğuna bağlayıp form seçim alanına ekleyeceğim.
